**AI Powered Health Prediction System**

A Flask-based web application that predicts health risks using Machine Learning.
This system enables healthcare providers to manage patient records efficiently while leveraging AI-based predictions to identify potential health risks such as Diabetes, Anemia, and Heart Disease.

**Project Overview**

The AI-Powered Health Prediction System combines patient management with predictive analytics to assist in early health risk assessment. By analyzing key health parameters such as Glucose, Haemoglobin, and Cholesterol levels, the system provides automated predictions and personalized recommendations to support preventive healthcare.

**Key Features**

* AI-Based Health Risk Prediction
* Diabetes Risk Detection
* Anemia Risk Detection
* Heart Disease Risk Detection
* Patient Record Search Functionality
* Add Patient

* Chatbot Integration 
* PDF Health Report Generation

* Responsive Bootstrap User Interface

* Personalized Health Recommendations

**Machine Learning Integration**

The application utilizes a trained Machine Learning model to analyze patient health metrics and classify potential health conditions. Predictions are generated in real time and stored alongside patient records for future reference.
