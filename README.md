# AI Powered Health Prediction System

A Flask-based web application that predicts health risks using Machine Learning.

## Features

- Add Patient
- Edit Patient
- Delete Patient
- Search Patient
- AI Health Prediction
- Diet Chart Recommendation PDFs
- SQLite Database

## Technologies

- Python
- Flask
- SQLite
- Scikit-Learn
- Bootstrap

## Run

pip install -r requirements.txt

python model.py

python app.py
